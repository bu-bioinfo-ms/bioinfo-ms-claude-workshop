# Agentic Coding for Bioinformatics — Claude Code Workshop

A hands-on workshop introducing **agentic coding workflows** to Bioinformatics MS
students, using [Claude Code](https://claude.com/claude-code) applied to real
scientific and bioinformatics tasks.

## Objective

Give students practical, hands-on experience running an AI coding agent as a
collaborator on scientific projects — not as an autocomplete, but as a teammate
you direct, verify, and course-correct.

## What we cover

- **Driving an agent** — framing tasks, planning before doing, and iterating.
- **Verification discipline** — never trust "it's done"; confirm it yourself
  (tests run, changes landed, results reproduce).
- **Memory & context** — how the agent accumulates project knowledge over time,
  and why it's just transparent, version-controlled files.
- **Bioinformatics-flavored tasks** — data wrangling, pipeline scaffolding,
  analysis scripts, and reproducibility.

## Guiding principles (the workflow we teach)

These are distilled from real day-to-day agentic work and form the backbone of
the workshop:

1. **Verify before claiming.** "Done" means *you checked* — tests pass, code
   landed, output reproduces. Not "the agent said so."
2. **Plan first, then act.** Agree on the approach before writing code; draft
   deliverables before implementing when asked to.
3. **Iterate empirically.** Measure, keep what helps, revert what hurts, and be
   willing to stop.
4. **Keep the human in the loop at decision points.** The agent proposes; you
   decide the science.
